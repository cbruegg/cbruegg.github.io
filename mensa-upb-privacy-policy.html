(ENGLISH - German version found below / Deutsche Version im Folgenden)

Mensa UPB automatically reports crashes to the developer using the Crashlytics service provided by Google, Inc. These crash reports include an ID that uniquely identifies each user to track the number of users affected by an issue. This data is not used for any other purposes.

(GERMAN / DEUTSCH)

Mensa UPB sendet automatisierte Berichte über Abstürze der App mithilfe des Dienstes Crashlytics von Google, Inc. an den Entwickler. Diese Absturzberichte enthalten eine ID, die den Benutzer eindeutig identifiziert, um die Anzahl der Benutzer, die von einem Problem betroffen sind, zu zählen. Die Daten werden zu keinem anderen Zweck verwendet.

Der Entwickler kann wiefolgt kontaktiert werden:
